🆕 5/12 미팅 반영 신규 기능 (2026-05-21)

1. 거래처 관리 UI 통일 개선
기존 "거래처 현황" 페이지가 경영지원팀 거래처 관리와 동일한 폼으로 교체되었습니다. 상단에 진료과별 카드 + 영업 통계 유지, 하단에 통합 거래처 관리 UI (검색 / 유형 필터 / 카드 / 매트릭스 / 배송지 / 등록). 영업 · 경영지원 · 품질관리팀 모두 동일한 UI 로 거래처를 보고 관리합니다.
2. 영업사원 마스터 계정 페이지 신규
좌측 사이드바 "영업사원 계정" 메뉴 신설. 영업사원(박진우 · 배경동 · 신현호) 카드에 본인 담당 거래처 수 · 4월 매출 · 목표 달성률 · 🎁 인센티브(초과분 10%) 표시. 카드 클릭 시 담당 거래처별 매출 / 발주 수 + "발주" 버튼으로 거래처 화면 즉시 이동.
3. 거래처별 할인율 매트릭스 신규
거래처 카드의 "할인 매트릭스" 버튼 → 리코탭플러스 / 리스프린트 / 실린더 / 바로웰핏 / 알티네오 모델별 할인율 + 자동 공급가 계산. 예: 리코탭플러스 상지 36,620원 × 55% = 20,141원 (실시간 갱신). "거래처별 할인율 단가.xlsx" 자료 그대로 반영.
4. 공지사항 신규
좌측 사이드바 "공지사항" 메뉴 (경영지원팀에서 작성). 거래처 발주 화면에서 로그인 시 최신 공지 팝업. 전체 / 대리점 / 병원 / 특정 업체별 타겟 발송.
5. 베트남 발주 프로젝트 추적 신규
좌측 사이드바 "베트남 발주 프로젝트" — 대표님 요청. 월별 발주 → 항공 ✈️ / 선박 🚢 분할 입고 트래킹 + 진행률 + 타임라인 뷰. 원단 / 부자재 / 제품 3분류.
6. UDI 공급내역 보고 신규
의료기기통합정보시스템(udiportal.mfds.go.kr) 보고 화면. 월 선택 → 병원 납품 자동 집계 → "UDI 전송" 버튼. ※ 실제 OAuth2 API 연동은 백엔드 단계 예정 — 현 화면은 mock
7. 매뉴얼 · 양식 카탈로그 + AI 챗봇 신규
ISO 13485 품질경영 매뉴얼 (1건) + 절차서 (16건) + 양식 (34건) 통합 조회. "교육훈련 절차 알려줘" 같은 자연어 질문 → 관련 문서 자동 추천.
8. 빠른 월 선택 (1월 ~ 12월 pill 버튼) 개선
조회 페이지에서 "1월 / 2월 / ... / 12월 / 이번달 / 올해" 버튼 클릭 → 시작일 / 종료일 자동 채움.
9. 알림 종 🔔 신규
상단 우측 종 모양 아이콘 — 미확인 알림 개수 표시 (빨간 점). 경영지원팀 발주 수정 · 베트남 입고 · 거래명세서 재발행 · 새 공지 등 실시간 알림.
10. 거래처 발주 취소 · 포장 박스 정보 신규
거래처 포털에서 발주 상태 화면에 "발주 취소 요청" 버튼 (포장 단계 전까지 가능). 포장 완료 시 박스 수 / 무게 / 송장번호가 거래처에 노출됨.
11. 매입매출장 발주 수정 + 이력 신규
경영지원팀이 발주 수량 변경/반품 처리 시, 변경 전/후 + 사유 + 작성자 자동 기록. 품질팀 칸반 카드에 ⚠️ 수정 알림 표시 — 확인 전까지 다음 단계 진행 차단.
12. 모달 표시 개선 개선
상세보기 / 매트릭스 / 등록 폼 등 모든 모달이 화면 안에 정확히 들어오도록 수정. 본문이 길면 모달 안에서만 스크롤 (화면 밖으로 안 나감).

  • 학회 방명록 입력 항목에 추가가 필요한 부분
  • 현장에서 모바일로 사용할 때 꼭 필요한 기능 (앱이 아닌 모바일 웹 기준)
