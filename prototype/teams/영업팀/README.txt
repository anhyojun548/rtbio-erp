[ 빠른 시작 ]

  1. 압축을 푸세요 (이미 풀려있으면 건너뛰기).
  2. "index.html" 파일을 더블클릭하세요.
  3. 크롬 또는 엣지 브라우저에서 자동으로 열립니다.

  → 설치, 회원가입, 인터넷 연결 모두 필요 없습니다.


[ 파일 구성 ]

  index.html        ← 메인 페이지 (이 파일을 더블클릭)
  사용설명서.html    ← 기능 안내 + 피드백 요청 항목
  README.txt        ← 이 파일
  css/              ← 스타일 시트 (8개)
  js/               ← 자바스크립트 + Mock 데이터 (10개)
  assets/           ← 로고 이미지


[ 이번 버전 주요 변경사항 ]

  ✓ 거래처 관리 UI 통일 (경영지원/영업/품질 동일 폼)
  ✓ 영업사원 마스터 계정 + 인센티브 계산 (NEW)
  ✓ 거래처별 할인율 매트릭스 (NEW)
  ✓ 공지사항 / 베트남 발주 트래킹 / UDI / 매뉴얼 (NEW)
  ✓ 알림 종 🔔 + 빠른 월 선택 (1~12월)
  ✓ 모달 표시 개선 (화면 안에 정확히 들어옴)

  자세한 내용은 [ 사용설명서.html ] 을 참고하세요.


[ 피드백 요청 ]

  실제 영업 업무와 비교하며 1-2일 사용해보신 뒤
  아래 항목들을 회신 부탁드립니다.

  • 대시보드에 꼭 필요한 지표 / 위젯
  • 영업사원 계정 인센티브 룰이 실무와 맞는지
  • 거래처 관리에서 추가 필요 항목
  • 학회 방명록 추가 입력 항목
  • 현장 모바일 사용 시 필요 기능
  • 누락 / 불편 / 꼭 필요한 기능
