📊 영업팀 프로토타입

알티바이오 ERP 시스템 · 영업팀 담당자용 사용설명서

📢 이번 버전에서 달라진 점

1. 대시보드 커스터마이징 기능 도입
각자 파악하고자 하는 데이터만 선택해서 화면에 표시할 수 있도록 개선했습니다.
담당자별 매출, 거래처, 품목별 판매 등 원하는 위젯을 자유롭게 배치할 수 있습니다.
2. 학회 방명록 개별 입력 기능 추가
학회 카드를 클릭하면 방문자를 한 명씩 등록/관리할 수 있습니다. (이름 / 연락처 / 소속 / 담당자 배정 / 접촉 상태 / 비고)
3. 담당자별 매출 품목 드릴다운
담당자 행을 클릭하면 해당 담당자의 품목 / 부위 / 사이즈별 판매 내역으로 바로 이동합니다.
4. 거래처 발주서 / 거래명세표 출력 버튼 추가
거래처 상세 모달에서 바로 출력할 수 있습니다.

🔑 주요 기능

① 대시보드

  • 위젯 자유 배치
  • 담당자별 / 품목별 / 거래처별 매출 현황
  • 원하는 데이터만 선별하여 본인 업무 화면 구성

② 거래처 관리

  • 거래처 상세 모달에서 발주서 / 거래명세표 출력
  • 담당자 배정 및 이력 관리
  • 거래처 유형에 "기타" 옵션 추가

③ 담당자별 매출 현황

  • 담당자 행 클릭 → 해당 담당자의 품목 / 부위 / 사이즈 드릴다운
  • 특정 담당자의 판매 패턴 분석 가능

④ 학회 / 거래처 방문 관리

  • 학회 카드 클릭 → 방명록 상세 모달
  • 방문자 개별 등록 (이름 / 연락처 / 소속 / 담당자 배정 / 접촉 상태 / 비고)
  • 방문자별 후속 영업 상태 추적

⑤ 사용량 입력

  • 거래처 현장 방문 시 품목 사용량 기록

▶️ 실행 방법
