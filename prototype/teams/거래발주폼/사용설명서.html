📋 거래처 발주폼 프로토타입

알티바이오 ERP 시스템 · 거래처 담당자용 사용설명서

📢 이번 버전에서 달라진 점

발주내역 조회 시 기간 필터가 추가되었습니다.
이번달 / 1개월 / 3개월 / 전체 / 직접설정(달력) 중에서 선택해서 원하는 기간의 발주 이력만 조회할 수 있습니다.

🔑 주요 기능

① 발주 신청

  • 품목 선택 및 수량 입력
  • 배송지 / 납기 요청일 지정
  • 비고란 작성 가능

② 발주내역 조회

  • 기간별 발주 이력 확인 (달력 기반 필터)
  • 각 발주 건의 진행 상태 확인 (접수 / 준비중 / 출고완료)

③ 거래처 정보

  • 담당자 연락처, 결제 조건 등 확인
