🔬 품질관리팀 프로토타입

알티바이오 ERP 시스템 · 품질관리팀 담당자용 사용설명서

📢 이번 버전에서 달라진 점

1. 대시보드 커스터마이징 기능 도입
각자 파악하고자 하는 데이터만 선택해서 화면에 표시할 수 있도록 개선했습니다. 위젯 추가/삭제/크기 조절이 자유롭습니다.
2. 칸반 보드 전반 개편
· "보드 수정" 버튼으로 단계(열) 추가/삭제/이름 변경 가능 (최대 8개)
· 각 단계마다 담당자를 따로 지정할 수 있습니다.
· 다음 단계로 넘길 때 담당자 지정 모달이 뜹니다.
3. 출고내역 페이지 추가
출고 완료된 건의 전체 히스토리와 각 단계별 담당자를 확인할 수 있습니다.

🔑 주요 기능

① 출고관리 칸반 보드

  • 드래그앤드롭으로 카드 단계 이동
  • 다음 단계 이동 시 담당자 지정 모달 팝업 (기본값 = 현재 담당자, 드롭다운 변경 가능)
  • 각 단계별 담당자 배지 표시 (카드 접힌 상태에서도 확인 가능)
  • 날짜 필터: 어제 / 오늘 / 이번주 / 기간선택
  • 카드별 거래명세서 PDF 출력 버튼 (동봉용)
  • 보드 수정 버튼으로 단계 구성 자유 변경 (최대 8개)

② 출고내역

  • 완료된 출고 건의 전체 히스토리 조회
  • 각 단계별로 담당했던 사람 확인 가능
  • 접수일자 / 출고일자 표시

③ 재고현황

  • 제품별 재고 확인
  • 제품 삭제 기능 (확인 모달 후 제거)

④ 샘플출고

  • 대리점 자동완성 검색

⑤ 설정

  • 업무 시작/종료 시각 직접 설정
  • 택배 마감시간 설정 (팀 상황에 맞게 자유 설정)

▶️ 실행 방법

같은 폴더 안의 index.html 파일을 더블클릭하시면 웹브라우저에서 바로 열립니다.
별도 설치나 인터넷 연결 없이 동작합니다.

  • 칸반 보드의 단계 구성이 실제 업무와 맞는지
  • 담당자 지정 방식이 편리한지
  • 거래명세서 출력 양식에서 수정이 필요한 부분
  • 누락된 항목이나 꼭 필요한 기능
  • 업무시간 / 택배 마감시간 외 추가 설정 필요 여부
