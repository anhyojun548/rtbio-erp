1. 스키마 개요

📁 public (공용)

  • User — 유저 (포털 전체 공용 · role 별 분기)
  • Tenant — 테넌트 메타
  • AuditLog — 모든 변경 감사 로그
  • SystemSetting — 시스템 전역 설정

📁 tenant_altibio (도메인)

  • Product · ProductSize · ExpiryLot
  • Client · ClientAddress · ClientDiscount · ClientFixedPrice
  • Order · OrderItem · Shipment · KanbanColumn ...
  • Invoice · Payment · ClosingLedger · BankTransaction
  • Conference · SalesContract · DataUsage · DashboardWidget
  • Inventory · InventoryLog · InventoryAdjustment
  • Notification · EmailQueue · TenantSetting · SalesAssignment
설계 의도 — 멀티테넌트 확장 시 tenant_{id} 스키마를 복제. 공용 유저는 public 에, 도메인 데이터는 테넌트별 격리. User.clientId 는 크로스 스키마 FK 를 피하기 위해 앱-레벨 참조.
