7. 아직 미구현 (운영 전환용)

📧 이메일 발송

EmailQueue 스키마만 있음. Daum SMTP 연동·발송 워커 미구현.

🔔 인앱 알림

Notification 스키마만 있음. 벨 아이콘·읽음처리 UI 미구현.

📎 파일 업로드

계약 PDF·제품 이미지 업로드 없음. Azure Blob 연결 필요.

📱 모바일 반응형

Desktop 우선. 테이블·사이드바 모바일 가공 미완.

🧪 E2E 테스트

Playwright 시나리오 미작성. 현재는 단위·스모크만.

📜 감사 조회 UI

AuditLog 기록만. 조회 화면 없음.

🏢 멀티테넌시 확장

현재 altibio 1곳. 테넌트 자동 생성 없음.
