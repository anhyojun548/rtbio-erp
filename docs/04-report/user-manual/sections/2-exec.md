## 2. 영업 포털 사용법

이 장은 RTBIO ERP의 **영업 포털**(영업팀 전용 화면)을 처음 쓰시는 분을 위한 사용 설명서입니다. 거래처를 관리하고, 매출 현황을 살펴보고, 담당자를 배정하고, 학회 방명록을 입력하고, 사용량과 보고서를 작성하는 일상 업무를 화면 순서대로 따라 하실 수 있도록 안내합니다.

### 들어가기 전에 — 로그인과 화면 구조

웹 브라우저(크롬 권장) 주소창에 **http://localhost:3000/exec** 를 입력해 접속합니다. 로그인 화면이 나오면 영업팀 계정(예: `sales1@rtbio.com`)과 **관리자가 안내한 비밀번호**를 입력하고 들어오시면 됩니다.

로그인하면 화면은 크게 두 영역으로 나뉩니다.

- **왼쪽 사이드바**: 위쪽에 로고와 내 이름(`김영업` · 영업팀)이 보이고, 그 아래에 메뉴가 세로로 나열됩니다. 메뉴는 "영업" 묶음과 "기타" 묶음으로 나뉘어 있습니다.
- **오른쪽 본문**: 선택한 메뉴의 내용이 표시되는 넓은 작업 공간입니다.

사이드바 메뉴를 클릭하면 오른쪽 본문이 해당 화면으로 바뀝니다. 지금 보고 있는 메뉴는 파란색으로 강조됩니다.

사이드바 내 이름 바로 아래에는 작은 글씨로 두 개의 링크가 있습니다.

- **🔑 비밀번호 변경**: 내 로그인 비밀번호를 바꾸는 작은 창을 엽니다. 현재 비밀번호와 새 비밀번호를 입력해 변경합니다.
- **↩ 로그아웃**: 포털에서 나갑니다. 공용 PC를 쓰신다면 자리를 비울 때 꼭 눌러 주세요.

사이드바 맨 아래에는 **모드** 드롭다운이 있습니다. 여기서 "관리자 (팀장) / 박진우 (팀장) / 신현호 (주임) / 이경동 (사원)" 중 하나를 고르면, 그 사람의 시야로 화면이 바뀝니다. **관리자(팀장)**를 고르면 모든 담당자의 거래처와 실적이 전부 보이고, 특정 사원을 고르면 그 사원이 담당하는 거래처와 실적만 보입니다. 팀장은 전체를 보고, 사원은 자기 것만 보는 구조라고 생각하시면 됩니다.

> 팁: 모드를 바꾸면 화면 우측 하단에 "모드 전환: ○○○" 같은 작은 알림(토스트)이 잠깐 떴다 사라집니다. 이 토스트는 저장·삭제·제출 등 거의 모든 동작 직후에 결과를 알려 주므로, 동작이 제대로 됐는지 확인하는 신호로 활용하세요.

그러면 사이드바 메뉴 순서대로 화면을 하나씩 살펴보겠습니다.

### 2.1 영업 대시보드

가장 먼저 보이는 첫 화면입니다. 우리 팀의 이번 달 영업 성과를 한눈에 요약해 주고, 원하는 정보 위젯을 직접 골라 배치할 수 있는 **나만의 현황판**입니다.

**화면 구성**

- 맨 위에 **"영업 대시보드"** 제목이 있고, 그 옆에 점 세 개 모양의 **⋮ 더보기 버튼**, 오른쪽 끝에 **➕ 위젯 추가** 버튼이 있습니다.
- 그 아래 가로로 **4장의 큰 숫자 카드(KPI)**가 놓여 있습니다.
  - **월 매출 현황**: 이번 달 발행된 거래명세서 합계 금액과 기준 월(예: 2026-04 누적).
  - **월 수금액 현황**: 이번 달 실제로 받은 돈의 합계.
  - **월 미수금 현황**: 아직 못 받은 돈의 합계와, 미수금이 있는 거래처 개수.
  - **담당 거래처**: 활성 거래처 총 개수와 대리점/병원 분포.
- 그 아래 넓은 영역이 **위젯 격자판(그리드)**입니다. 여기에 매출 차트, 거래처 순위표 같은 위젯들이 카드 형태로 놓입니다.

**위젯을 추가해 보겠습니다**

오른쪽 위 **➕ 위젯 추가** 버튼을 누르면 "위젯 추가" 창이 열립니다. 이 창은 두 부분으로 되어 있습니다.

1. **🤖 AI에게 말하기**: 위쪽 채팅 칸에 "이번 달 거래처별 매출 보여줘" 처럼 평소 말로 입력하면, 그에 맞는 위젯을 추천해 줍니다(AI 연동이 켜진 환경에서 동작).
2. **갤러리에서 선택**: 아래쪽에 미리 만들어 둔 위젯들이 카드로 나열됩니다. 원하는 카드를 클릭하면 대시보드에 바로 추가됩니다.

갤러리 영역 오른쪽 위의 **+ 직접 만들기** 버튼을 누르면 "위젯 직접 만들기" 창이 열려, 원하는 위젯을 처음부터 구성할 수 있습니다. 이 창에서 채우는 항목은 다음과 같습니다.

- **위젯 제목**: 위젯에 표시될 이름(예: "이번 달 매출").
- **데이터 소스**: 어떤 데이터를 쓸지(주문, 거래처 등).
- **차트 종류**: 막대·표 등 어떤 모양으로 보여줄지.
- **측정값(집계)**: 무엇을 합산·평균낼지(합계/개수 등 + 대상 항목).
- **분류(그룹)**: 어떤 기준으로 묶을지(거래처별, 제품별 등).
- **필터 조건**: **+ 조건** 버튼으로 "어떤 행만 볼지" 조건을 추가합니다.
- **정렬 기준 / 정렬 / 최대 행**: 큰 값부터(내림차순)인지 작은 값부터(오름차순)인지, 몇 줄까지 보여줄지.

설정하는 동안 오른쪽 **미리보기** 영역에 실제 데이터가 즉시 그려집니다. 마음에 들면 아래 **저장**을, 그만두려면 **취소**를 누릅니다.

**만든 위젯을 정리해 보겠습니다**

대시보드에 놓인 위젯은 **마우스로 끌어(드래그)** 위치를 옮기거나 크기를 조절할 수 있습니다. 각 위젯에 마우스를 올리면 점 세 개 또는 우클릭으로 작은 메뉴가 뜨는데, 여기서 다음을 할 수 있습니다.

- **✏ 위젯 수정**: 제목을 바꿉니다(구성 자체를 바꾸려면 "직접 만들기"로 새로 만들라는 안내가 함께 나옵니다).
- **복제**: 같은 위젯을 하나 더 만듭니다.
- **새로고침**: 최신 데이터로 다시 그립니다.
- **삭제**: 위젯을 대시보드에서 없앱니다.

**대시보드 전체를 관리해 보겠습니다**

제목 옆 **⋮ 더보기** 버튼을 누르면 메뉴가 펼쳐집니다.

- **대시보드 내보내기**: 지금 구성을 파일로 저장합니다(백업·이동용).
- **대시보드 가져오기**: 저장해 둔 구성 파일을 불러옵니다.
- **기본 레이아웃으로 초기화**: 내가 바꾼 배치를 모두 지우고 처음 상태로 되돌립니다.

> 주의: **기본 레이아웃으로 초기화**는 지금까지 꾸며 둔 위젯 배치를 한 번에 되돌립니다. 공들여 만든 구성이 있다면 먼저 **대시보드 내보내기**로 백업해 두시길 권합니다.

대시보드로 현황을 파악했다면, 이제 기간과 담당자별로 더 자세히 들여다보겠습니다. 왼쪽 **영업 현황** 메뉴로 이동합니다.

### 2.2 영업 현황

선택한 기간 동안의 매출을 **병원/대리점으로 나눠 보고, 담당자별 실적과 품목별 판매까지 파고드는** 분석 화면입니다.

**화면 구성**

- 맨 위 **"영업 현황"** 제목 아래에 회색 카드 형태의 **기간 필터 막대**가 있습니다.
  - **시작일 / 종료일**: 날짜를 직접 고릅니다(처음 들어오면 최근 7일이 기본으로 채워져 있습니다).
  - **유형** 드롭다운: 전체 / 병원만 / 대리점만 / 기타만 중에서 고릅니다.
  - **조회** 버튼: 위에서 고른 조건으로 다시 계산합니다.
  - 오른쪽 끝의 **오늘 / 이번 주 / 이번 달** 버튼: 자주 쓰는 기간을 한 번에 채워 줍니다.
- 그 아래 **4장의 KPI 카드**: 총 매출 / 병원 매출 / 대리점 매출 / 신규 거래처.
- 그 아래 두 개의 그래프 카드: 왼쪽 **병원 매출**, 오른쪽 **대리점 매출**(거래처별 막대그래프).
- 그 아래 **담당자별 매출** 표.
- 맨 아래 **품목/부위/사이즈 드릴다운** 영역.

**기간을 정해 조회해 보겠습니다**

예를 들어 이번 달 실적을 보려면, 오른쪽의 **이번 달** 버튼을 누르면 됩니다. 그러면 시작일·종료일이 이번 달로 채워지고 화면이 즉시 갱신됩니다. 특정 구간만 보려면 **시작일**과 **종료일**을 직접 고른 뒤 **조회** 버튼을 누릅니다. 병원만 따로 보고 싶다면 **유형** 드롭다운에서 "병원만"을 고르세요. 제목 바로 아래 작은 글씨에 "(시작일) ~ (종료일) · (선택한 유형)"이 표시되어, 지금 어떤 조건으로 보고 있는지 알려 줍니다.

**담당자별 실적을 보겠습니다**

화면 가운데 **담당자별 매출** 표에는 담당자마다 한 줄씩, 다음 정보가 들어 있습니다.

- **담당자 / 역할**: 이름(동그란 머리글자 아이콘)과 직급.
- **담당 거래처**: 맡고 있는 거래처 수.
- **발주 건수 / 매출액**: 기간 내 발주 건수와 매출 금액.
- **반품건수**: 반품으로 처리된 건수(있을 때만 노란 배지로 표시).
- **신규**: 새로 튼 거래처 수(초록 배지).

이 표의 **각 줄을 클릭**하면, 그 담당자를 기준으로 맨 아래 품목 드릴다운이 다시 계산됩니다. 선택된 줄은 색이 강조되고 "● 선택됨" 표시가 붙습니다. 같은 줄을 한 번 더 클릭하면 선택이 풀려 전체 기준으로 돌아갑니다.

> 팁: **모드** 드롭다운에서 특정 사원을 골라 두면, 이 표에서 그 사원 외의 다른 담당자 줄은 흐릿하게 표시되어 본인 실적에 집중할 수 있습니다.

**품목까지 파고들어 보겠습니다**

맨 아래 **품목/부위/사이즈 드릴다운**에는 브랜드(예: 리코트 플러스, 스프린트)별로 접힌 줄이 있고, 오른쪽에 그 브랜드의 총 매출이 표시됩니다. 브랜드 줄 왼쪽의 **▶ 화살표**를 누르면 줄이 펼쳐지며(▼로 바뀜), 그 안의 **부위(상지/하지/무릎 등)와 사이즈(S/M/L/XL)별 판매 수량·금액**이 표 형태로 나타납니다. 다시 누르면 접힙니다.

위에서 담당자 줄을 선택해 둔 경우, 드릴다운 위쪽에 "○○○ 담당 기준 품목 드릴다운 (담당 비중 N%)"이라는 안내 띠가 뜨고 그 담당자 기준으로 숫자가 보정됩니다. 띠 안의 **✕ 필터 해제** 버튼을 누르면 다시 전체 기준으로 돌아갑니다.

실적을 파악했으니, 실제 거래처 정보를 손볼 차례입니다. 왼쪽 **거래처 관리** 메뉴로 이동합니다.

### 2.3 거래처 관리

우리 거래처의 정보를 **조회·등록·수정**하고, 거래처별 **할인율·고정 단가**를 설정하며, 거래처 상세에서 **발주서·거래명세표를 출력**하는 화면입니다. 이 화면은 경영지원팀이 쓰는 화면과 동일한 형태입니다.

**화면 구성**

- 맨 위 **"거래처관리"** 제목 아래에 검색 막대가 있습니다.
  - **거래처명 검색** 입력칸: 글자를 입력하면 즉시 목록이 걸러집니다.
  - **전체 유형** 드롭다운: 전체 / 대리점 / 병원 / 기타.
  - **+ 신규 거래처 등록** 버튼(파란색).
- 그 아래 거래처들이 **카드 격자** 형태로 나열됩니다. 각 카드에는 거래처명·유형 배지, 담당자, 영업담당, 연락처, 이메일, 주소, 결제방식, 마감기간, 명세서 종류, 할인 태그, 고정단가가 표시됩니다.
- 각 카드 맨 아래에 **수정 / 할인 매트릭스 / 품목별 할인 / 상세보기** 버튼이 있습니다.

**거래처를 찾아보겠습니다**

검색 칸에 거래처 이름의 일부(예: "한빛")를 입력하면 해당 거래처 카드만 남습니다. 유형으로 추리려면 **전체 유형** 드롭다운에서 "병원" 등을 고르세요. 검색과 유형은 함께 적용되며, 위쪽에 "총 거래처 N개" 요약이 갱신됩니다.

**새 거래처를 등록해 보겠습니다**

오른쪽 **+ 신규 거래처 등록** 버튼을 누르면 등록 창이 열립니다. 거래처명·유형·담당자·연락처·주소·결제방식 등 기본 정보를 채우고 저장하면, 새 거래처 카드가 목록에 추가됩니다.

**기존 거래처를 수정·설정해 보겠습니다**

각 카드 아래 버튼의 역할은 다음과 같습니다.

- **수정**: 거래처의 기본 정보(연락처·주소·담당자 등)를 바꾸는 창을 엽니다.
- **할인 매트릭스**: 브랜드(모델)별 정가 대비 공급률(할인율)을 한 표에서 일괄 설정합니다. 각 칸에 % 값을 넣으면 공급가가 자동 계산됩니다.
- **품목별 할인**: 카테고리(무릎/상지/하지/스프린트) 기본 할인율 위에, 개별 제품마다 다른 할인율을 따로 지정합니다. 비워 두면 카테고리 기본값이 적용됩니다.
- **상세보기**: 거래처 상세 정보 창을 엽니다(아래에서 자세히 설명).

**거래처 상세를 열어 보겠습니다**

**상세보기**를 누르면 그 거래처의 종합 정보 창이 뜹니다. 담당 영업사원, 결제 조건, 미수금, 누적 매출, 의사 수, 신규 거래일, 지역·주소, 담당자 연락처가 정리되어 있고, 아래로 **취급 품목**, **품목별 판매수량(이번 달)** 표, **경쟁업체 품목(비고)** 메모칸이 이어집니다.

- **경쟁업체 품목(비고)** 칸에 메모를 적으면 거래처 정보에 함께 저장됩니다. 메모칸 밖을 클릭하거나 창 아래 **비고 저장** 버튼을 누르면 저장됩니다.
- 창 맨 아래에는 **발주서 출력**과 **거래명세표 출력** 버튼이 있습니다. 각각 새 창에 인쇄용 양식을 띄워, 그 창의 **인쇄** 버튼으로 종이 출력하거나 PDF로 저장할 수 있습니다.

> 주의: **발주서 출력·거래명세표 출력**은 새 창(팝업)으로 열립니다. 브라우저가 팝업을 막으면 "팝업을 허용해주세요" 알림이 뜨니, 주소창 옆의 팝업 차단 아이콘에서 이 사이트의 팝업을 허용해 주세요.

거래처 정보를 정리했다면, 이제 그 거래처들을 영업사원에게 나눠 맡길 차례입니다. 왼쪽 **담당자 관리** 메뉴로 이동합니다.

### 2.4 담당자 관리

영업사원마다 **어떤 거래처를 맡고 있는지** 배정하고 해제하는 화면입니다.

**화면 구성**

- 맨 위 **"담당자 관리"** 제목 옆에 **+ 거래처 배정** 버튼이 있습니다.
- 그 아래에 영업사원별 **카드**가 격자로 놓입니다. 각 카드에는 사원 이름·직급·담당 거래처 수가 있고, 그 아래에 담당 거래처가 한 줄씩 나열됩니다.

**거래처를 배정해 보겠습니다**

오른쪽 위 **+ 거래처 배정** 버튼(또는 각 사원 카드 안의 **+ 거래처 배정** 버튼)을 누르면 배정 창이 열립니다.

1. **담당자** 드롭다운에서 거래처를 맡길 사원을 고릅니다.
2. **거래처 선택** 드롭다운에서 배정할 거래처를 고릅니다(이미 배정된 곳은 목록에 나오지 않습니다).
3. 저장하면 해당 사원 카드에 거래처가 추가되고, "○○ → ○○○에게 배정" 알림이 뜹니다.

**배정을 해제하려면**, 사원 카드의 담당 거래처 줄 오른쪽에 있는 **✕** 버튼을 누릅니다. "○○를 담당에서 제외할까요?" 확인 후 제외됩니다.

> 권한 안내: 거래처를 배정·해제하는 **✕ 버튼과 배정 버튼은 관리자(팀장) 모드에서만 보입니다.** 사이드바 **모드**가 특정 사원으로 되어 있으면 본인 카드만 조회 형태로 보이고, 배정·해제 버튼은 나타나지 않습니다. 배정 작업을 하려면 모드를 "관리자(팀장)"로 바꿔 주세요.

담당자를 정리했으니, 학회 현장에서 모은 잠재 고객을 입력해 보겠습니다. 왼쪽 **학회 관리** 메뉴로 이동합니다.

### 2.5 학회 관리

참가한 **학회별 성과(방문자·접촉·신규거래·성공률)**를 관리하고, 학회 현장에서 받은 **방명록(방문자 명단)**을 입력해 영업으로 연결하는 화면입니다.

**화면 구성**

- 맨 위 **"학회 관리"** 제목 옆에 **+ 학회 등록** 버튼이 있습니다.
- 그 아래 **4장의 KPI 카드**: 총 방명록 / 접촉 완료 / 신규 거래처 / 미접촉.
- 그 아래 **최근 학회** 표: 학회명, 종류, 날짜, 부스비, 설치비, 방명록(방문자), 접촉, 신규거래, 성공률, 담당자 배분이 한 줄씩 표시됩니다.

**새 학회를 등록해 보겠습니다**

오른쪽 위 **+ 학회 등록** 버튼을 누르면 등록 창이 열립니다.

- **학회명**: 예) 대한정형외과학회 2026 추계.
- **종류**: 정형외과 / 신경외과 / 재활의학 / 기타.
- **날짜**: 학회 일자(오늘 날짜가 기본으로 채워져 있습니다).
- **방명록 건수**: 예상 방문자 수(예: 100).

저장하면 표 맨 위에 새 학회가 추가되고, 방문자 수는 입력값을 세 담당자에게 균등 배분합니다(배분은 이후 조정 가능).

**방명록을 입력해 보겠습니다**

표에서 **학회명(파란 밑줄 글씨)을 클릭**하면 그 학회의 **방명록 창**이 열립니다. 위쪽 "➕ 방문자 추가" 영역에서 다음을 입력합니다.

- **이름**(필수), **연락처**, **소속**(필수, 병원/기관명).
- **담당자** 드롭다운: 이 방문자를 누가 팔로우업할지.
- **상태** 드롭다운: 미접촉 / 접촉완료 / 팔로우업 / 신규계약 / 거절.
- **비고**: 메모(선택).

**추가** 버튼을 누르면 아래 명단 표에 한 줄이 더해지고, 입력칸이 비워져 다음 사람을 바로 입력할 수 있습니다. 명단의 각 줄 오른쪽 **삭제** 버튼으로 잘못 넣은 항목을 지울 수 있습니다. 방문자를 추가/삭제할 때마다 위쪽 학회 KPI(접촉·신규 등)가 자동으로 다시 계산됩니다. 입력을 마쳤으면 창 아래 **닫기** 버튼으로 나갑니다.

> 참고: 학회 표의 **성공률**은 "신규거래 ÷ 접촉" 으로 계산됩니다. 즉 접촉한 사람 중 몇 %가 실제 거래로 이어졌는지를 보여 줍니다. 방문자 상태를 "신규계약"으로 정확히 바꿔 두면 성공률이 올바르게 집계됩니다.

학회 명단을 정리했으니, 이제 사원별 매출과 인센티브를 확인해 보겠습니다. 왼쪽 **영업사원 계정** 메뉴로 이동합니다.

### 2.6 영업사원 계정

영업사원별 **담당 거래처·월별 매출·목표 달성률·인센티브**를 한눈에 보고, 그 자리에서 바로 발주 화면으로 넘어갈 수 있는 화면입니다.

**화면 구성**

- 맨 위 **"영업사원 마스터 계정"** 제목.
- 그 아래 사원별 **카드**가 격자로 놓입니다. 카드마다 이름·이메일, 담당 거래처 수, 이번 달 매출, **목표 달성률 막대**가 있고, 목표를 초과한 사원은 "🎁 인센티브: ○○원 (초과분 10%)" 띠가 추가로 표시됩니다.
- 카드 아래 **선택한 사원 상세** 영역: 고른 사원의 월별 매출 추이·월 목표·연락처와, 담당 거래처별 4월 매출/발주 건수 표가 나옵니다.

**사원을 골라 상세를 보겠습니다**

사원 카드를 **클릭**하면 아래 상세 영역이 그 사원 기준으로 채워집니다(처음 들어오면 첫 번째 사원이 자동 선택됨). 상세 표의 각 거래처 줄에는 거래처 코드·이름·유형·진료과·4월 매출·발주 건수가 보이고, 맨 오른쪽에 **발주** 버튼이 있습니다.

**그 자리에서 발주해 보겠습니다**

거래처 줄의 **발주** 버튼을 누르면, 그 거래처 계정으로 **발주 화면이 새 창으로 열립니다**(거래처 포털). 영업사원이 거래처를 대신해 주문을 넣어 줄 때 사용합니다.

> 참고: 인센티브는 "월 매출이 월 목표를 초과한 경우, 초과분의 10%"로 계산되어 카드에 표시됩니다. 목표 달성률 막대는 100% 이상이면 초록, 70~99%는 주황, 그 미만은 빨강으로 색이 달라져 한눈에 진척도를 알 수 있습니다.

여기까지가 "영업" 묶음 메뉴입니다. 이제 "기타" 묶음의 공지·사용량·보고서·데이터 도구를 살펴보겠습니다. 왼쪽 **📢 공지사항** 메뉴로 이동합니다.

### 2.7 공지사항

거래처에 보낼 **공지(자료 안내, 혜택 안내 등)를 작성·관리**하는 화면입니다. 영업팀 이름으로 발송됩니다.

**화면 구성**

- 맨 위 제목 옆에 **＋ 공지 작성** 버튼이 있습니다.
- 그 아래 **제목/본문 검색** 입력칸으로 기존 공지를 빠르게 찾을 수 있습니다.
- 그 아래 공지 목록이 나오며, 각 공지마다 **미리보기 / 편집 / 삭제** 버튼이 있습니다.

**공지를 작성해 보겠습니다**

**＋ 공지 작성** 버튼을 누르면 작성 창이 열립니다.

- **제목**(필수), **긴급도**(일반 / 긴급).
- **본문**(필수).
- **발송 대상** 드롭다운: 전체 거래처 / 대리점만 / 병원만 / **특정 업체**.
  - "특정 업체"를 고르면 아래에 업체 선택 영역이 펼쳐집니다. **검색창**으로 업체를 찾고 체크해서 고르거나, **검색결과 전체선택 / 전체해제** 버튼으로 한꺼번에 조정할 수 있습니다. 고른 업체는 위쪽에 칩(태그)으로 표시되며, 칍의 제거 버튼으로 빼낼 수 있습니다.
- **만료일**(선택): 이 날짜가 지나면 공지가 자동으로 내려갑니다.
- **상단 고정(📌 표시)** 체크박스: 중요한 공지를 목록 맨 위에 고정합니다.

작성자는 **영업팀**으로 자동 설정됩니다. 아래 **저장** 버튼으로 발송하면 됩니다.

**기존 공지를 다루겠습니다**

목록의 각 공지 버튼은 다음과 같습니다.

- **미리보기**: 거래처에게 보일 모습을 그대로 확인합니다.
- **편집**: 내용을 고칩니다(이 버튼은 편집만 하며 삭제하지 않습니다).
- **삭제**: 공지를 없앱니다(빨간색).

> 주의: **삭제**는 되돌릴 수 없습니다. 잠시 안 보이게만 하려면 삭제 대신 **만료일**을 과거 날짜로 설정하거나, 임시로 내용을 비워 두는 방식을 고려하세요.

공지를 발송했다면, 다음으로 거래처의 실사용 수량을 입력해 보겠습니다. 왼쪽 **사용량 입력** 메뉴로 이동합니다.

### 2.8 사용량 입력

거래처(주로 병원)의 **이번 달 실제 사용 수량**을 입력해 월 정산의 기초 자료를 만드는 화면입니다.

**화면 구성**

- 맨 위 **거래처 선택** 드롭다운(병원만 표시됨)과 **마감 기간**(읽기 전용 표시)이 있습니다.
- 그 아래 **사용량 입력 표**: 제품명·사이즈·전월사용·**당월사용(입력칸)**·단가·소계.
- 표 아래 **임시저장 / 제출** 버튼.
- 그 아래 **품목별 사용량·판매액(전체 거래처 합계)** 요약표.
- 맨 아래 **이전 제출 내역** 표.

**사용량을 입력해 보겠습니다**

1. **거래처 선택** 드롭다운에서 입력할 병원을 고릅니다. 그러면 그 거래처의 마감 기간과 품목 목록이 표에 채워집니다.
2. 각 줄의 **당월사용** 입력칸에 이번 달 실제 사용 수량을 넣습니다. 숫자를 바꾸면 그 줄의 **소계**와 표 아래 **합계**가 즉시 다시 계산됩니다.
3. 작성 중간에 보관하려면 **임시저장**, 최종 확정하려면 **제출** 버튼을 누릅니다.

**제출**을 누르면 "사용량을 제출하시겠습니까? 제출 후 수정이 제한됩니다." 확인 창이 뜨고, 확인하면 저장됩니다.

> 주의: **제출 이후에는 수정이 제한됩니다.** 숫자를 한 번 더 점검한 뒤 제출하시고, 작성 도중에는 **임시저장**을 활용하세요. 저장이 끝나면 우측 하단에 "임시 저장되었습니다 / 사용량이 제출되었습니다" 알림이 뜹니다.

아래쪽 **품목별 사용량·판매액** 요약표는 전체 거래처의 사용량을 제품·규격별로 합산해, 전월 대비 증감(▲▼)과 월 판매액을 보여 줍니다. **이전 제출 내역** 표에서는 과거에 어떤 거래처를 어느 기간에 제출했는지 확인할 수 있습니다.

사용량 입력을 마쳤다면, 품질 양식 보고서를 작성해 보겠습니다. 왼쪽 **보고서 작성** 메뉴로 이동합니다.

### 2.9 보고서 작성

알티바이오의 **ISO 13485 품질 양식**을 골라 채우고, AI 어시스턴트의 도움을 받아 빠르게 작성한 뒤 **인쇄·PDF로 출력**하는 화면입니다.

**화면 구성**

이 화면은 좌우 두 영역으로 나뉩니다.

- **왼쪽(양식 영역)**: 맨 위 **양식 선택** 드롭다운으로 작성할 품질 양식을 고르면, 아래에 그 양식의 입력 폼/미리보기가 나타납니다. 양식 아래에는 작업 버튼이 있습니다.
- **오른쪽(AI 어시스턴트)**: 자연어로 지시하면 양식을 자동으로 채워 주는 채팅 영역입니다.

**양식을 작성해 보겠습니다**

1. 왼쪽 위 **양식 선택** 드롭다운에서 작성할 양식을 고릅니다. 그러면 해당 양식이 화면에 펼쳐집니다.
2. 항목을 직접 입력해도 되고, 오른쪽 AI 채팅 칸에 예를 들어 "한빛정형외과로 발주서 작성" 처럼 입력한 뒤 **↑(전송)** 버튼을 누르면 AI가 관련 칸을 자동으로 채워 줍니다. 채팅 아래의 추천 문구(suggestions)를 눌러 빠르게 시작할 수도 있습니다.

**작성한 양식을 저장·출력하겠습니다**

왼쪽 양식 아래 버튼은 다음과 같습니다.

- **임시저장**: 작성 중인 내용을 보관합니다.
- **양식대로 인쇄**: 양식 서식 그대로 인쇄 창을 엽니다.
- **PDF 출력**: 작성한 양식을 PDF 파일로 내보냅니다.

보고서를 출력했다면, 마지막으로 원본 데이터를 직접 들여다보는 도구를 살펴보겠습니다. 왼쪽 **데이터 탐색기** 메뉴로 이동합니다.

### 2.10 데이터 탐색기

거래처·제품·주문 등 **원본 데이터를 표 형태로 직접 조회·필터·편집하고, CSV로 내보내는** 고급 도구입니다.

**화면 구성**

- 맨 위 **테이블 선택** 드롭다운으로 어떤 데이터(거래처·제품·주문 등)를 볼지 고릅니다. 오른쪽에는 작업 버튼(권한이 있으면 **＋ 새 행**, 그리고 **CSV** 내보내기)이 있습니다.
- 그 아래 **필터 막대**와 **요약 줄**.
- 그 아래 데이터가 **표(그리드)**로 표시되고, 맨 아래에 **페이지 이동** 영역이 있습니다.

**데이터를 조회하고 내보내 보겠습니다**

1. **테이블 선택** 드롭다운에서 보고 싶은 데이터를 고릅니다. 표 위에는 그 테이블의 이름과 설명이 표시됩니다.
2. **필터 막대**에서 조건을 걸어 원하는 행만 추립니다. 표 아래 페이지 버튼으로 다음/이전 페이지를 넘깁니다.
3. 화면에 보이는(필터가 적용된) 데이터를 파일로 받으려면 오른쪽 위 **CSV** 버튼을 누릅니다. "CSV N행 내보냄" 알림과 함께 파일이 저장됩니다.

**데이터를 편집하겠습니다(권한이 있을 때)**

- **＋ 새 행** 버튼을 누르면 오른쪽에 입력 패널이 열려, 새 데이터를 추가할 수 있습니다(아래 **추가** 버튼으로 저장).
- 기존 행을 클릭하면 같은 패널이 **편집** 모드로 열려, 값을 고친 뒤 **저장**할 수 있습니다.

> 주의: 데이터 탐색기는 원본 데이터를 직접 바꾸는 강력한 도구입니다. **＋ 새 행** 버튼이나 편집 패널은 해당 권한이 있을 때만 나타나며, 여기서의 추가·수정은 실제 데이터에 바로 반영됩니다. 단순히 자료를 보거나 받기만 할 때는 **CSV 내보내기**만 사용하시고, 편집은 내용을 정확히 확인한 뒤 신중히 진행하세요.

### 마무리

여기까지가 영업 포털의 모든 화면입니다. 일상 업무는 보통 **영업 대시보드**로 현황을 살피고 → **영업 현황**에서 기간·담당자별로 분석하고 → **거래처 관리**에서 정보와 단가를 손보고 → **담당자 관리**로 거래처를 배정하고 → **학회 관리**로 잠재 고객을 모으고 → **사용량 입력**으로 정산 자료를 만드는 흐름으로 이어집니다. 필요할 때 **공지사항**으로 거래처에 안내하고, **보고서 작성**으로 품질 양식을 출력하며, **데이터 탐색기**로 원본을 직접 확인하시면 됩니다.

작업 중 헷갈릴 때는 화면 우측 하단에 잠깐 뜨는 **알림(토스트)**으로 결과를 확인하시고, 공용 PC에서는 끝나고 꼭 **로그아웃**해 주세요.
