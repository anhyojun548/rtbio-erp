7. 필수 규칙 (Invariants)

  • 모든 tenant_{id} 테이블은 미들웨어에서 강제된 테넌트 컨텍스트를 통해서만 접근한다.
  • 주문 확정 이후 OrderItem.unitPrice는 변경 불가 (immutable).
  • 재고 변동은 반드시 InventoryLog 행을 생성한 뒤 Inventory를 갱신한다 (트랜잭션 보장).
  • 모든 DML은 AuditLog에 before/after JSON으로 기록한다.
  • CASCADE 정책은 Prisma 모델 주석에 명시 — Order 삭제 시 OrderItem CASCADE, Product 삭제 시 RESTRICT.
