📊 장단점 정리

각 클라우드 선택 시 트레이드오프

AWS

장점

  • ✅ 과업내용서 명시된 스택 (별도 협의 불필요)
  • ✅ 국내 개발자 풀 가장 넓음
  • ✅ 서비스 종류가 가장 많음
  • ✅ 국내 레퍼런스 풍부 (문제 해결 빠름)
  • ✅ Next.js 생태계 최적 (SST, Amplify)

단점

  • ❌ 콘솔/IAM 복잡, 학습 곡선 가파름
  • ❌ 크레딧 확보 어려움
  • ❌ 데이터 전송 비용 Azure보다 비쌈
  • ❌ 기본값이 관대함 (보안 실수 위험)
Azure

장점

  • ✅ 크레딧 $6,000 확보 용이
  • ✅ Microsoft 생태계 (M365, GitHub, Copilot)
  • ✅ 콘솔 직관적, IAM 간단
  • ✅ 엔터프라이즈 선호 (대기업/공공)
  • ✅ 기본 보안 설정 보수적 (실수 방지)

단점

  • ❌ 과업내용서 변경 필요
  • ❌ 국내 AWS 대비 개발자 풀 좁음
  • ❌ 일부 서비스 AWS보다 설익음
  • ❌ Next.js 전용 도구 AWS만큼 풍부하지 않음
