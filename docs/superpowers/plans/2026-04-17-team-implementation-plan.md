### 팀간 데이터 파이프라인

```
[거래처] 발주 작성
    ↓
[QC팀] 발주확정 → 가용재고 차감 → 칸반 진입
    ↓  (접수대기→피킹→검수→포장→출고대기→출고완료)
    ↓
[경영지원] 거래명세서 자동생성 → 수금 관리
    ↓
[영업팀] 매출 대시보드 반영 → 담당자별 실적
    ↓
[CEO] 통합 현황
```
